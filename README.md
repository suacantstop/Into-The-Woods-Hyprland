## 🍃 О проекте
Этот райс - гармония между строгой декларативностью **NixOS** и уютной эстетикой темного леса. Система построена на базе **Nix Flakes** и использует централизованное управление цветами через `palette.nix`. 

## ⌨️ Основные горячие клавиши (Binds)
Клавиша-модификатор по умолчанию: `CAPSLOCK` 

| Сочетание | Действие |
| :--- | :--- |
| `MOD3 + T` | Запустить основной терминал (**Foot**) |
| `MOD3 + B` | Открыть браузер (**Firefox**) |
| `MOD3 + R` | Запустить меню приложений (**Fuzzel**) |
| `MOD3 + Q` | Закрыть активное окно |
| `MOD3 + M` | Выйти из Hyprland |
| `MOD3 + F` | Переключить полноэкранный режим |
| `MOD3 + 1-6` | Переключение между воркспейсами |
| `MOD3 + Shift + 1-6` | Переместить окно на воркспейс |
| `MOD3 + H/J/K/L` | Перемещение фокуса (Vim-style) |
